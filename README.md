# LookaGenius Platform

Educational platform project with premium Glassmorphism design, 3D interactive background, and dynamic animations.

## Features
- **Premium UI**: Dark Slate & Violet theme inspired by Flutter UI.
- **Interactive Background**: Three.js particle network background that reacts to mouse movement.
- **Dynamic Content**: Typewriter effect for slogans and course quotes.
- **Responsive Animations**: AOS (Animate On Scroll) integration for a smooth scrolling experience.
- **Multi-Category**: Sections for Languages, Sciences, Engineering, ICT, and detailed Course listings.
- **Scholarships**: Dedicated section for European scholarship opportunities (Erasmus, DAAD, etc.).

## Project Structure
- `index.html`: Main entry point.
- `style.css`: Custom CSS styles.
- `script.js`: Custom JavaScript logic.

## Libraries Used
- [Three.js](https://threejs.org/)
- [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/)
- [Font Awesome](https://fontawesome.com/)
- [Google Fonts](https://fonts.google.com/)

## How to Run
Simply open `index.html` in any modern web browser.
